安装指南

按照以下步骤快速部署您的Termux Nginx服务器

0

准备工作: 安装Termux sv服务

pkg install termux-services -y
1

安装Termux

从F-Droid或GitHub下载并安装Termux应用

2

更新软件包

pkg update && pkg upgrade
3

安装Nginx

pkg install nginx
4

启动Nginx服务

nginx
5

管理Nginx服务

sv (stop | disable | start | status | enable) nginx
6

访问服务器

在浏览器中访问 http://localhost:8080
